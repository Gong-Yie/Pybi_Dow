# Pybi_Dow

在找素材的时候，发现b站的视频下载很不友好，于是Pybi_Dow出现了。
这是一个基于 `yt-dlp` 和 `PySide6` 开发的带图形界面的 Bilibili 视频下载工具。

## 功能特点

- **交互式界面**：告别黑框命令行，操作直观。
- **智能解析预览**：输入链接一键解析，自动显示视频封面、标题、UP主和视频时长。
- **画质自由选择**：支持 1080p、720p、480p 等多种分辨率以及“最高画质”自动匹配。
- **自定义保存路径**：可视化选择下载文件夹。
- **多线程防卡死**：解析和下载全部在后台线程运行，界面流畅不假死，实时显示下载进度条。
  
## 启动

在根目录下执行下面的命令

```bash
python main.py
```

## 运行环境与安装

### 安装 Python

请确保你的电脑上安装了 **Python 3.8 或以上版本**。安装时请务必勾选 `Add Python to PATH`。

### 安装依赖库

```bash
pip install -r requirements.txt
```
